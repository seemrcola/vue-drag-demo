`当前支持的快捷键`
- 画布拖拽 `SHIFT + 鼠标左键`   
- 画布缩放 `CTRL + 鼠标滚轮`
- 多选    `CTRL + 鼠标左键`
- 复制粘贴 `CTRL + c / v`
- 上下左右 `⬆️ ⬅️ ⬇️ ➡️`  
- 删除    `DEL`  
- 撤销重做 `CTRL + z / y`
- 组件锁定与解锁 `CTRL + l / o`
